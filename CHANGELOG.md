## 1.0.0
# 🌟 Custom Text Field Pro

![Pub Version](https://img.shields.io/pub/v/custom_text_field_pro)
- Added `onCountryCodeChanged` callback to retrieve the selected country code.
- Added `titleStyle`, `bottomSheetHeight`, `pickerRadius`, and `pickerDecoration` to `CustomTextFieldTheme` for extensive customization.
- Added `prefixIconConstraints` and `suffixIconConstraints` properties to control icon box sizes.
- Removed default splash effect from prefix and suffix icons for a cleaner UI.
- Improved text overflow handling for the selected country code.
- Fixed controller disposal logic to prevent memory leaks when no external controller is provided.
